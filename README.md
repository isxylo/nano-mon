# nano-mon

个人轻量级服务器监控面板，一键部署到 Linux 服务器。

## 功能

- **nano-mon**：实时系统监控面板，显示 CPU、内存、网络、进程等信息
- **nano-fb**：双栏文件浏览器，左栏显示根目录一级子目录，右栏显示选中目录内容，支持键盘和鼠标操作
- **dash**：入口命令，启动完整的 tmux 布局（监控面板 + 文件浏览器）

## 依赖

- bash
- tmux
- curl
- nano
- tput（ncurses）

## 安装

```bash
bash nano-mon
```

执行后自动检测并安装依赖，将 `nano-mon`、`nano-fb`、`dash` 写入 `/usr/local/bin/`。

## 使用

```bash
dash
```

启动后进入 tmux 会话，布局如下：

```
┌─────────────────────────────────┐
│                                 │
│         nano（编辑区）           │
│                                 │
├───────────────┬─────────────────┤
│  文件浏览器    │   系统监控面板   │
│  (nano-fb)   │   (nano-mon)    │
└───────────────┴─────────────────┘
```

## nano-fb 键位

| 按键 | 功能 |
|------|------|
| ↑ ↓ | 移动选择 |
| Tab | 切换左/右栏焦点 |
| Enter | 进入目录 / 用 nano 打开文件 |
| Backspace | 返回上级目录 |
| / g G | 跳转到指定路径 |
| q | 退出 |
| 鼠标滚轮 | 滚动列表 |
| 鼠标单击 | 选中条目 |
| 鼠标双击 | 进入目录 / 打开文件 |
